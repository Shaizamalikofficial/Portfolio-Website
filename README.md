A clean personal website to showcase projects skills and experience. Built with Next.js TypeScript Tailwind CSS and Framer Motion. Content is organized into focused sections that highlight work impact and core skills.

## Features
- Animated hero with subtle glow effects
- About section with goals and background
- Projects grid with tech tags and key metrics
- Skills section in a single clean column
- Vertical timeline for learning milestones
- Contact section with direct links

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started
Prerequisites
- Node.js 18 or newer

Install and run
```bash
npm install
npm run dev
```
App runs on `http://localhost:3000`

## Available Scripts
- `npm run dev` starts the dev server
- `npm run build` creates a production build
- `npm start` runs the production server

## Environment Variables
If you use Contentful add these variables. Remove any you do not need.

```
CONTENTFUL_SPACE_ID=
CONTENTFUL_ENVIRONMENT=master
CONTENTFUL_CDA_TOKEN=
CONTENTFUL_PREVIEW_TOKEN=
```

## Deploy
This project deploys best on Vercel. Connect the GitHub repository import the project set environment variables then deploy. Custom domains can be added from Vercel settings.

## Project Structure
```
portfolio/
  app/                # Routes and pages
  components/         # UI components
  lib/                # Data and utilities
  public/             # Static assets
  tailwind.config.ts  # Tailwind setup
```
